Track Your Finances Securely

A comprehensive system for managing income, expenses, and budgets with secure user authentication and database integration.

Income Tracking

Record and categorize all sources of income. Monitor earnings over time with detailed reports and visualizations.

Expense Management

Log daily expenses across customizable categories. Set spending limits and receive notifications when approaching budgets.

Budget Planning

Create monthly budgets for different categories. Compare actual spending against planned budgets with visual indicators.

Project Requirements

This application fulfills the CIS 344 project requirements including:

  • Database joins for relational data
  • Transaction management for data integrity
  • Responsive HTML/CSS interface
  • Input validation and security measures
